# Color Style – Summary

**Theme:** White + Deep Tech Blue
**Direction:** Bright · Clean · Modern · Premium · Tech-focused
**Use Case:** Electronics E-commerce

## Brand Accent (Primary)

* Primary: `#1E40AF`
* Hover / Active: `#1E3A8A`
* Soft Background: `#EEF2FF`
* Usage: Primary buttons, CTAs, price highlight, links, active icons
* Rule: **Do not use for long-form body text**

## Background

* Page: `#F9FAFB`
* Card / Surface: `#FFFFFF`

## Text Colors

* Heading / Title: `#111827`
* Body Text: `#374151`
* Muted / Secondary: `#6B7280`

## Border / Divider

* Border color: `#E5E7EB`

## Semantic Colors

* Error: `#DC2626` (soft: `#FEF2F2`)
* Success: `#16A34A` (soft: `#ECFDF5`)
* Warning: `#F59E0B` (soft: `#FFFBEB`)
* Rule: Use strictly by semantic meaning only

## Usage Principles

* White dominates ~70–80% of UI
* Brand blue used strictly as accent
* CTAs should stand out but not feel saturated
* Prefer soft background variants for highlights

## Visual Personality

* Modern
* Clean
* Premium
* Tech-oriented
* Responsive-ready (desktop & mobile)
