# Typography System – Summary

## Font Family

* **Primary:** Inter
* **Fallback:** `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, Arial, sans-serif`

## Font Weights

* Regular – 400
* Medium – 500
* SemiBold – 600
* Bold – 700

## Typography Scale

### Headings

* H1: 24px / 600
* H2: 20px / 600
* H3: 18px / 600
* H4: 16px / 600

### Body

* Large: 16px / 400
* Medium: 14px / 400
* Small: 13px / 400
* Caption: 12px / 400

### UI Elements

* Button: 14px / 500
* Input: 14px / 400
* Label: 13px / 500
* Badge: 12px / 600

## Rules

* Minimum font size: 12px
* Prefer 500 over 700 for emphasis
* Keep hierarchy consistent
* Avoid mixing too many font sizes in one component
